# Coffee Shop app
A fullstack coffee shop app with React Native Expo, Nodejs, Express and MongoDB

<img src="https://github.com/jpdoshi/coffee-shop-app/blob/main/cover.png?raw=true" style="width: 100%; display: block" />

## Project Setup
Open terminal in directory and type following commands:

```bash
# clone repo
git clone https://github.com/jpdoshi/coffee-shop-app.git

# install dependencies
npm install

# start server
npm run start
```
